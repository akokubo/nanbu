アプリについて

「戦国南部氏の城めぐり」アプリは、八戸市博物館令和3年度夏季特別展「乱世の終焉－根城南部氏と城－」開催にあたり、八戸市博物館と八戸工業大学の共同研究で開発しました。

本アプリは、中世から近世にかけて北東北に勢力を誇った南部氏ゆかりの城館をマップと一覧表で確認できるアプリです。

コンテンツは、八戸市博物館が制作しました。

掲載されている城館は、天正20年6月11日付「南部大膳大夫分国之内諸城破却共書上」に記された48城館を中心に、関連城館を加えたものです。「南部大膳大夫分国之内諸城破却共書上」は、『新編八戸市史　中世資料編』所収の「南部古今録」三（遠野市立博物館所蔵赤沢家文書）を底本としています。

プログラムは、八戸工業大学工学部システム情報工学科小久保温研究室が、Google Maps APIを用いて、JavaScriptで開発しました。

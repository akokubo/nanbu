諸城破却とは

天正18年7月、小田原北条氏を滅ぼした豊臣秀吉は、宇都宮城に入城し、奥羽の大名に対して領地仕置を行った。7月〜8月にかけて行われたこの仕置を、奥羽仕置という。三戸南部氏南部信直は、7月27日付で豊臣秀吉から南部領内7郡の所領安堵の朱印状を受けている。この時、合わせて家中の諸城破却と妻子の三戸居住が命じられた。

これを受け、天正20年6月11日付で提出されたのが、「南部大膳大夫分国之内諸城破却共書上」である。領内の48城のうち36城が破却、12城が残された。信直の居城であった福岡城はこれに含まれていない。

原本は不明であり、写のみ残る。

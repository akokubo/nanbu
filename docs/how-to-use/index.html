使い方

①地図・リストからお城をさがす

「お城リスト」をクリックし、地図上のポイント又は表のお城の名前をクリックすると、お城の個別説明の画面に移ります。

ブラウザの戻るボタンか、「お城リスト」をクリックすると、一覧表に戻ります。

②「諸城破却書上」に記載されたお城を見る

「諸城破却とは」をクリックすると、「諸城破却書上」に記されたお城の一覧表と地図が表示されます。表のお城の名前をクリックすると、お城の個別説明の画面に移ります。
